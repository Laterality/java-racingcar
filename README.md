# 프리코스 2주차 미션 - 자동차 경주

우아한테크코스 프리코스 2주차 미션을 구현한 프로젝트입니다.

## 실행

1. 프로젝트 복제

    ```
    git clone https://github.com/Laterality/java-racingcar
    ```

2. 실행

    선호하는 IDE로 프로젝트를 열고 `AppMain` 클래스를 실행합니다.

## 예시 입출력

```
경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
John,May,Jim

시도할 회수는 몇회인가요?
4

실행결과
John : -
May : -
Jim : -

John : --
May : -
Jim : -

John : ---
May : --
Jim : -

John : ---
May : ---
Jim : --

John, May가 최종 우승했습니다.
```
